# ATM

ATM stands for "ATM is Terminal Multiplexer"

## Installation

```bash
npm install -g @jcubic/atm
```

## Usage

```bash
atm
```

## Development

Install bun:

```bash
npm install -g bun
```

Clone the repository:

```bash
git clone https://github.com/jcubic/atm.git
```

To install dependencies:

```bash
bun install
```

To run:

```bash
bun start
```
